# Contributing to TSun FreeFire JWT API

## Ownership
This project is owned by **TSun Studio** and **Script Kittens**.
No contributions are accepted without prior written permission from the owners.

## Contribution Policy
- Do not submit pull requests, patches, or issues unless you have been explicitly invited to do so.
- All code, documentation, and other contributions are under the exclusive control of TSun Studio and Script Kittens.
- Unauthorized copying, redistribution, or use of project content is prohibited.

## Requesting Permission
If you wish to contribute, request permission from the project owner first.
Provide:
- Your name or organization
- A description of the proposed contribution
- Contact information

Only after receiving explicit approval may you submit contributions.

## Reporting Issues
You may report bugs or issues only through the communication channels approved by TSun Studio.
Do not use reported issues as permission to reuse or redistribute the project.
